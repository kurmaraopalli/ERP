A next-generation, highly modular Enterprise Resource Planning (ERP) application designed for enterprise capabilities with a zero-configuration footprint. This system utilizes a structured, transaction-safe **Flat-File JSON database architecture**, making it completely self-contained, light, and optimized for rapid deployment and edge testing.

---

## 🗺️ System Architecture & Data Flow

This ERP application divides core business functions into specialized, decoupled modules. Instead of a heavy relational database, modules read and write data via a centralized, asynchronous, transaction-safe I/O Database Engine mapping directly to JSON files.
